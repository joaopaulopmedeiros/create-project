# create-project

## 📑 Description
A tiny and simple create-project lib. Start coding with TS and JS on an easy way!

## ❓ How to use it?
- `git clone https://github.com/joaopaulopmedeiros/create-project.git`
- `npm install`
- `npm link`
- `mkdir <project_name> && cd <project_name>`
- `create-project`
- Open your favorite IDE

## Flags 
- `--git or --g`
